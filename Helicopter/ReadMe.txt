Helicopter Game - Orbi Web Games Developer : bootcamp
Made by: Guilherme Ferreira Medeiros Lossio
Basically, it's a helicopter game where the player must eliminate as many enemies as possible and rescue as many allies as possible. In this game you have 3 lives before you lose the game.
To improve performance I made it only move if it was in a zone within the limit instead of when it exceeds the limit it returns to the limit, thus making the gameplay smoother.
